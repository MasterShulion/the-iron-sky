✈️ О сервере

The Iron Sky — это уютный сервер Minecraft, где царит атмосфера творчества, дружбы и приключений. Мы играем с модами Create и Create Aeronautics, строим механизмы, летаем на дирижаблях и исследуем мир вместе.

Здесь нет жёстких правил и давления — только приятная атмосфера, где каждый может реализовать свои идеи.

🎮 Наши моды

Create — механика, транспорт, автоматизация

Create Aeronautics — авиация, дирижабли, летающие конструкции

Пиратские моды — интересные дополнения для приключений
